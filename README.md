# 🚀 Portfólio Oficial | Pedro Nichollas (PNP Systems)

Bem-vindo ao repositório do meu portfólio pessoal. Este projeto foi desenvolvido para centralizar os projetos da minha trajetória profissional, destacando projetos de alta complexidade em Desenvolvimento Web, Banco de Dados e consultoria Oracle ERP.

## ⚙️ Arquitetura e Diferenciais do Projeto

Este não é apenas um site estático comum, a arquitetura foi pensada com foco em escalabilidade e manutenção inteligente:

* **Single Source of Truth (SSOT):** Toda a alimentação de dados do site (projetos, habilidades, textos e contatos) é feita através de um único arquivo `dados.json`. 
* **Gerador de Currículo Dinâmico:** O sistema conta com um motor (`cv.html`) que consome o JSON principal e renderiza automaticamente um currículo limpo e formatado em padrão A4, pronto para impressão em PDF através da URL paramétrica `?action=print`.
* **Design "Dark Tech":** Interface construída com foco em UI/UX corporativo, utilizando efeitos de *Glassmorphism* e paleta de cores voltada para tecnologia.

## 🛠️ Stack Tecnológica

* **HTML5** (Semântico e estruturado)
* **CSS3** (Variáveis nativas, Flexbox, CSS Grid, media queries para responsividade)
* **Vanilla JavaScript** (ES6+, Fetch API, manipulação de DOM)
* **JSON** (Estruturação e persistência de dados no Front-end)

## 🌐 Projetos em Destaque no Portfólio

* **Nexus Enterprise ERP:** Sistema de gestão logística com módulos WMS/TMS e rastreamento.
* **Portal do Cidadão (e-Gov):** SPA focado na gestão pública e transparência.
* **Oracle Cost Manager v3:** Suite FinOps para monitoramento e alertas de orçamento.
